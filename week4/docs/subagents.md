## TestAgent (TDD Specialist)
- **Role:** Write comprehensive, failing tests before any code is written.
- **Constraints:** Do NOT modify any files in `backend/app/`. Only create or edit files in `backend/tests/`.
- **Primary Tool:** `pytest`.

## CodeAgent (Backend Engineer)
- **Role:** Implement the minimal code required to pass existing tests.
- **Constraints:** Follow Clean Code principles. Ensure all logic is inside `backend/app/`.
- **Primary Tool:** `fastapi`, `sqlalchemy`.
